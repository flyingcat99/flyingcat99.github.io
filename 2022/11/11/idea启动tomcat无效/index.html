idea启动tomcat无效

Application Server was not connected before run configuration stop, reason: Unable to ping server at localhost:1099

端口占用（情况一）

直接打开cmd窗口，将占用端口的进程杀死即可

catalina.bat文件中的设置(情况二)

Tomcat中配置的JAVA_OPTS将idea中的JAVA_OPTS覆盖,想要配置jvm参数可以直接在idea中配置
